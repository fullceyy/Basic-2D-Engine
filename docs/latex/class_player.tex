\doxysubsection{Detailed Description}
A class representing a player-\/controlled sprite in a game. 

The {\ttfamily \doxylink{class_player}{Player}} class extends {\ttfamily \doxylink{class_sprite_object}{Sprite\+Object}} to include frame animations for a player object, has implementation of input events for movement and animation. It manages direction the player is facing, travel speed, and idle state.
