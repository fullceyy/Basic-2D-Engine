\doxysubsection{Detailed Description}
A class representing a bitmap object with posibiliity of custom draw and transform behaviour. 

The {\ttfamily \doxylink{class_bitmap_object}{Bitmap\+Object}} class allows creating and managing a bitmap object that can be drawn, translated, rotated, scaled, and cropped. It integrates with SDL to render textures and performs transformations on the bitmap directly.

The bitmap\+Object is heavily reliant on the SDL\textquotesingle{}s source and destination rectangle. In the spritesheet, it will be setting a position of where the source rectangle should be reading and copying it into destination rectangle in the renderer.
