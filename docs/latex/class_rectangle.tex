\doxysubsection{Detailed Description}
A class representing a rectangle with drawing and transformation capabilities. 

The {\ttfamily \doxylink{class_rectangle}{Rectangle}} class provides methods to create a rectangle and perform transformations such as translation, rotation, and scaling. The rectangle is rendered using an {\ttfamily SDL\+\_\+\+Renderer} and a specified color.
