\doxysubsection{Detailed Description}
A class representing a 2D point that can be transformed and drawn. 

The {\ttfamily \doxylink{class_point2_d}{Point2D}} class virtually inherits from {\ttfamily \doxylink{classshape_obj}{shape\+Obj}}, which provides transformation and drawing methods. It represents a 2D point with specific x and y coordinates. The class provides methods for setting and getting the coordinates, as well as drawing and transforming the point when inherited behavior is used.
