\doxysubsection{Detailed Description}
Abstract base class for objects with animation capabilities. 

The {\ttfamily \doxylink{class_animated_object}{Animated\+Object}} class provides a pure virtual method {\ttfamily animate} to be implemented by derived classes. It serves as a base class for any object requiring animation functionality.
