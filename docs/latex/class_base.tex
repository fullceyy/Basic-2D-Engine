\doxysubsection{Detailed Description}
An abstract base class defining a common interface for update and draw operations, actual name Game\+Object. 

This class works as a base for other classes that require functionality, such as Drawing and Transforming an Object. Derived classes must provide implementations for the {\ttfamily update} and {\ttfamily draw} methods.
