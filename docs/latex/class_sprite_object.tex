\doxysubsection{Detailed Description}
A class representing an animated sprite object with drawable and animated capabilities. 

The {\ttfamily \doxylink{class_sprite_object}{Sprite\+Object}} class extends {\ttfamily \doxylink{class_bitmap_object}{Bitmap\+Object}} and {\ttfamily \doxylink{class_animated_object}{Animated\+Object}} to provide functionalities for drawing, loading and animating sprites. It uses an SDL texture and source rectangles to manage frame-\/based animations.
