\doxysubsection{Detailed Description}
This class will be handling fundamental bitmap operations such as creation, loading, deleting, saving and copying to other bmp. 

Will be using SDL\textquotesingle{}s SDL\+\_\+\+Surface structure to manipulate bitmap operations.
