Detailed Description

A class representing a player-controlled sprite in a game.

The Player class extends SpriteObject to include frame animations for a player object, has implementation of input events for movement and animation. It manages direction the player is facing, travel speed, and idle state.

void Player::inputEventHandler ( SDL_Event & event)
inline

Handles keyboard input events to control the player's movement and facing / walking direction.

Parameters
eventThe SDL_Event object containing keyboard input data.
  • SDL_KEYDOWN: Updates the direction and moves the player. Idle is set to false since the movement key is pressed. Each SDLK case which represents a direction, will call the setDirection() method and translate() by specified movespeed.
  • SDL_KEYUP: Sets the player to idle(true).
