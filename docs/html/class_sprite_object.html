Detailed Description

A class representing an animated sprite object with drawable and animated capabilities.

The SpriteObject class extends BitmapObject and AnimatedObject to provide functionalities for drawing, loading and animating sprites. It uses an SDL texture and source rectangles to manage frame-based animations.
