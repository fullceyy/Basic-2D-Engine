Detailed Description

A class representing a shape object that can be transformed and / or drawn. Inherits from Transformability and DrawAbility.

The shapeObj class is more specific implementation of the Transformability and DrawAbility classes. It combines the ability to perform geometric transformations (rotate, scale, translate) with the ability to draw the object. It overrides the virtual methods from both the Transformability and DrawAbility classes to provide specific behavior for each shape.
