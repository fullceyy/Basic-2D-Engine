Detailed Description

A class representing a 2D line segment between two points with drawing and transformation capabilities.

The LineSegment class represents a line segment defined by two Point2D objects: the start and end points. It provides functionality for setting / getting the points(their coordinates), drawing the segment from defined points, and applying transformations such as translation, rotation, and scaling. The class also uses an SDL_Renderer and an SDL_Color to render the line segment to a window with a given color.
